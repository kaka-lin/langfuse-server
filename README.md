# Self-Hosted Langfuse Server

A personal, self-hosted deployment of the [Langfuse](https://langfuse.com) AI observability platform. Built and optimized for local development across macOS and Linux environments.

## Components

- Langfuse Web & Worker (v3)
- PostgreSQL 16
- ClickHouse
- Redis 7

## Quick Start

We provide convenient scripts with automatic environment setup.

### Start the Server

```bash
./start.sh
```

### Stop the Server

```bash
./stop.sh
```
